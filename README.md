## 🎮 About CraftLand

CraftLand is an open-world multiplayer voxel game built for explorers, builders, warriors, and traders.

Venture into vast landscapes, uncover hidden treasures beneath the surface, construct towering creations, defend yourself against the darkness, and participate in a player-driven economy powered by Solana.

Every action contributes to a shared universe that evolves alongside its community.

---

## ✨ Features

### 🌍 Persistent World
A living multiplayer universe that continues to grow and evolve.

### ⛏️ Mining & Resource Gathering
Dig deep underground to discover rare materials and hidden treasures.

### 🏗️ Creative Building
Build homes, castles, villages, cities, or anything your imagination can create.

### ⚔️ Adventure & Combat
Fight hostile creatures, survive dangerous environments, and conquer challenges.

### 💎 Rare Loot & Treasures
Collect valuable resources, artifacts, and unique items.

### 🤝 Multiplayer Economy
Trade resources, equipment, and collectibles with other players.

### 🌲 Diverse Biomes
Explore forests, mountains, deserts, caves, and mysterious regions.

### ⚡ Solana Powered
Fast transactions, digital ownership, and scalable infrastructure.

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/craftland.git
```

### Enter Directory

```bash
cd craftland
```

### Install Dependencies

```bash
npm install
```

### Start Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
